## Editing the Project

You have multiple ways to edit this application:

### 2. Use Your Local IDE

Clone this repo and work locally. Changes pushed to the repo will also appear in Lovable.

**Requirements:** Node.js & npm ([install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating))

```sh
# Clone the repository
git clone https://github.com/Ne-x-tr-on/Davion.git

# Navigate to project directory
cd Davion

# Install dependencies
npm install

# Start development server with live preview
npm run dev
